配置chrome

现在虽然配置好了SS，但是还是无法打开网页的

需要配置chrome

  1. 下载Proxy SwitchyOmega
    我们需要下载一个chrome 浏览器的插件 Proxy SwitchyOmega，但是没有代理之前是不能从 Google 商店安装这个插件的，但是我们可以从 Github 上直接下载最新版：https://github.com/FelisCatus/SwitchyOmega/releases/

  2. 安装Proxy SwitchyOmega
    然后浏览器地址打开chrome://extensions/，将下载的插件托进去安装。

  3. 配置Proxy SwitchyOmega
    3.1 安装完成之后，我们会在浏览器的菜单栏看到一个蓝色环形小图标，点击选项按钮
    3.2 情景模式中选择 proxy ，在代理服务器中的代理协议选择 socks5，本地代理服务器 127.0.0.1，代理窗口1080。完成之后，一定一定要点击下面的应用选项，进行保存
