Tensorflow Basic 02 张量 Tensor

发表于 2018-03-10 | 分类于 Deep Learning ， Tensorflow

本文主要记录一下在学习TF时候的知识

tensorflow 有三个最基本的概念
分别是：

  1. 计算图 Graph
  2. 张量 Tensor
  3. 会话 Session

此篇重点介绍 张量

张量

张量，也就是tensor

是tensorflow中管理数据的形式

其实理解张量，需要一些基本的DL知识

要能分得清，标量，向量，N维数组等概念

重点来了，(这里需要思考一下)

张量在TF中保存的是如何得到这些数字的计算过程

可以看这个例子

可以看到，当我们返回一个tensor的时候，并没有得到a+b的结果(数字)，而是得到了一个看起来包含一些信息的结构

这就是张量的结构

第一个位置，可以看成是这个张量的唯一标示符，这个名字方便我们在debug状态或者tensorboard状态更方便的找到它

后面两个就是shape信息和type信息了

对了，每个张量都是有唯一的类型的，所以也需要参与张量计算的所有张量都是一个type，如果不是的话，会报错

有个小tip需要注意

我们应该尽量细的将计算拆分，保留每个步骤的张量，这样更加方便与日后debug
