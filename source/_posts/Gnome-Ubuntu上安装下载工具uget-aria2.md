这篇文章介绍如何在Gnome Ubuntu 16.04上安装类似迅雷的下载软件uget & aria2
<!--more-->
### 安装 uget & aria2
```
sudo apt-get update

sudo apt-get install uget

sudo apt-get install aria2

```
### 将 uget 的下载设置为 aria2 

open uget 
edit --> setting --> plug-in 
plug-in matching order: aria2

### 配置 firefox

进入附加组件管理页面  
搜索并安装flashgot
